       邮件控制远程电脑关机

**************************************

1、项目介绍：
   本项目基于python、shell脚本语言开发，类似黑客常用的守护技巧，另计算机不断监控邮箱的状态；

2、环境支持：
   运行本项目电脑中需有python环境支持；

3、第三方支持：
   邮件控制，所以需要有一个第三方邮箱账号，系统默认收件发件邮箱为sina邮箱。

4、邮箱配置支持：
   sina邮箱系统配置需支持smtp、pop服务器开启状态；

5、邮箱地址以及密码可随意更换；
